# tinychronicler-web

Statically hosted web interface for [Tiny Chronicler](../README.md).

## Requirements

- Node 16
- npm

## Development

```bash
# Install dependencies
npm install

# Build application in development mode whenever something changes
npm run watch

# Check linters
npm run lint

# Build application for production
npm run build
```
